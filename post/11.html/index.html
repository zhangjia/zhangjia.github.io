Linux标记

2019-08-16 4 min read

一. 命令

  • 命令格式：命令 参数 目标对象

    • ls -l /ur 以列表的方式查看usr文件下目录文件

  • cd：切换目录

    • -cd 回到家目录

      -cd~ 回到家目录

      -cd / 回到根目录

      -cd - 回到上次工作目录

      -cd . 当前目录

      -cd .. 回到上一级目录

  • ls：查看当前文件夹下的所有文件夹

    • ls -h 以单位显示大小
    • ls -l 列示文件和目录的详细信息，还有一种简介的写法：ll

  • pwd：查看当前工作目录的完整路径

  • df：查看当前磁盘使用情况

    • -h：使用K、M、G为单位显示
    • -a：显示所有文件系统的磁盘使用情况
    • -k：以k字节为单位显示
    • -T： 显示文件系统类型（T必须大写）
    • 无论在哪个文件下输入 df -h等命令，都会显示同样的信息，因为该命令显示的是当前磁盘的使用情况

  • du：查看当前目录空间使用情况

    • -h： 使用K、M、G为单位显示
    • -k：以KB(1024bytes)为单位输出
    • -m ：以MB为单位输出
    • -s：仅显示总计，只列出最后加总的值
    • 可以组合使用，比如打开usr文件夹，键入du -hs 或者du -sh 可以查看以GB为单位显示的总计

  • yum install tree：安装tree

  • su 用户名 ：切换用户

    • su root

  • mkdir ：创建一个或者多个目录，用-p递归创建目录

    • mkdir demo
    • mkdir a/b/c -p ：递归创建多级目录 mkdir -p a/b/c 也可以

  • touch：创建新文件

    • touch a.log

  • >： 创建新文件

    • > b.log

  • cp：复制

    • a目录下有a.log和b目录，将a.log，复制到b目录中：cd a 然后 cp a.log b
    • a目录下有a.log和b目录，将a.log，复制到b目录中，并改名为b.log：cd a 然后 cp a.log b/b.log
    • demo目录下有a目录，a目录下有b目录，b目录下有多个文件，将b目录复制到demo目录下和a目录同级：：cd demo/a 然后 cp b .. 会显示略过目录，因为b目录下还有文件，所以需要递归复制：cp -r b ..

  • mv：移动

    • 和上面的命令都一样，只不过把cp改成mv
    • 重命名：比如把a目录下的a.log改成ab.log：mv a.log ab.log

  • 删除

    • 删除a目录下的a.log：进入a目录后，rm a.log需要输入y确认才能删除，可以使用 rm -rf a.log来直接删除，不询问
    • 可以直接用多个空格分隔多个文件名，批量删除

  • 查看文件内容

    • cat 文件名

      • cat -n显示文件内容时，打印行号

      • cat –b同上，但排除空行

  • more 文件名：分屏查看，回车翻页，只能向下翻页，q或者ctrl + c退出

    • less 文件名：分屏查看，可以向上向下键翻页 ，只有q能退出

  • head 文件名：前十行

    • head -n x：显示文件头x行

    • tail 文件名：后十行

      • tail -n x：显示文件末尾x行

    • tail -f ：实时显示文件变化

  • 压缩

    • zip 压缩文件名 源文件 ：压缩文件
    • zip a.zip a.log
      • 或者直接写zip a a.log
    • zip -r 压缩文件名 源目录 ：压缩目录
      • zip -r a.zip a
    • unzip 压缩文件名 ：解压缩
      • unzip a.zip

  • 压缩

    • tar -zcvf 压缩包名.tar.gz 源文件

  • 选项

    • z : 压缩为tar.gz

    • 解压缩

      • tar -zxvf 压缩包名.tar.gz

      • 选项

      • x: 解压缩.tar.gz格式

    • 压缩多个文件：tar -zcvf test.tar.gz a.log b.log

    • 压缩目录不需要加-r，和压缩文件一样

  • rpm -qa查看当前系统安装了哪些软件

    • rpm -qa | grep gcc 查看gcc有没有安装

三. vim

操作模式

命令模式

控制屏幕光标的移动，字符、字或行的删除，移动复制某区段及进入Insert mode下，或者到 last line mode。

插入模式

只有在Insert mode下，才可以做文字输入，按「ESC」键可回到命令模式。

底行模式

将文件保存或退出vi，也可以设置编辑环境，如寻找字符串、列出行号……等。

基本命令

  • vim + 文件名：打开后光标位于文件最后一行
  • vim +n 文件名：打开后光标位于文件第n行
  • vim +/xxx 文件名：打开后光标位于xxx所在的行，可以用于快速定位指定内容
  • vim 文件a 文件b 文件c：打开（创建多个文件），底行模式“n”切换

四. Linux安装web环境

1. 安装Java

  1. yum search java :
  2. yum install java-1.8.0-openjdk ：安装jre
  3. yum install java-1.8.0-openjdk-devel ：安装jdk，直接安装这个也可以，上面的那个会被自动安装
  4. 安装的时候加-y参数可以不用输入y安装，直接安装

2. 安装mysql

  1. yum search mariadb
  2. yum install mariadb-server
  3. 启动sql：systemctl start mariadb
  4. 设置用户名和密码：mysqladmin -u root password zhangjia （不用登陆）

3. 允许mysql的root用户远程登录

GRANT ALL PRIVILEGES ON *.* TO 'root'@'%' IDENTIFIED BY 'zhangjia';
GRANT ALL PRIVILEGES ON *.* TO 'root'@'localhost' IDENTIFIED BY 'zhangjia';
FLUSH PRIVILEGES;

# 其中by后面的密码就是yml文件中配置的密码，yml中的密码要随着这个密码而更改，还可以直接用mysqladmin来设置这个by后面的密码
# 这三个命令是登录下

4. 安装Tomcat

  1. 打开安装目录：cd /usr/local

  2. 下载Tomcat：wget http://mirror.bit.edu.cn/apache/tomcat/tomcat-8/v8.5.43/bin/apache-tomcat-8.5.43.tar.gz

  3. 解压缩安装包：tar -zxvf apache-tomcat-8.5.43.tar.gz

  4. 启动Tomcat：进入bin目录，输入./startup.sh启动tomcat

    4.1关闭tomcat：`./showdown.sh`

  5. 关闭防火墙：systemctl stop firewalld.service

  6. ifconfig查看ip

5. 部署SpringBoot的应用程序

  1. 将项目打包
    mvn package
  2. 导出数据
    使用Navicat导出SQL文件
  3. 导入数据
    使用Navicat导入SQL文件
  4. 将打包好的jar文件上传到服务器
    使用xftp上传文件
  5. 在服务器启动jar文件
    java -jar xxx.jar
    后台启动：nohup java -jar xxx.jar &
  6. 通过进程结束程序
    ps -ef | grep java，可以看到进程的id
    kill 进程id

6. 自动联网

  1. cd /etc/sysconfig/netword-scripts
  2. vim ifcfg ，将ONBOOT修改为yes

7. maven打包

  1. mvn clean 清除clean
  2. mvn package 打包

8. 开机自启的服务

  1. 检查服务当前状态：systemctl status xx
  2. 启动服务：systemctl start xx
  3. 停止服务：systemctl stop xx
  4. 关闭防火墙：systemctl stop firewalld.service
  5. 启动防火墙：systemctl start firewalld.service
  6. 禁止防火墙开机自启：systemctl disable firewalld.service
  7. 开启MariaDB ：systemctl enable mariadb

五. docker

1. 安装docker

  • 查看当前系统的内核版本：uname -r

  • 使用yum安装docker

    • yum search docker
    • yum install docker

  • 启动docker：systemctl start docker

  • 停止docker：systemctl stop docker

  • 检查版本：docker -v

2. 使用docker安装Tomcat镜像

  • docker serarch tomcat
  • docker pull tomcat :下载Tomcat镜像
  • 启动Tomcat镜像：docker run --name tomcat1 -p8080:8080 -d tomcat
    • docker run --name tomcat2 -p8081:8080 -d tomcat
    • docker run --name tomcat3 -p8082:8080 -d tomcat

3. docker常用命令

  • 查看已经安装的镜像：docker images
  • 查看正在运行的容器：docker ps
    • 查看所有容器：docker ps -a
  • 停止指定容器：docker stop 容器id
  • 启动指定容器：docker start 容器id
  • 删除指定容器：docker rm 容器id
  • 重启指定容器：docker restart 容器id
