命令

  • cd：切换目录

    • cd / ：打开根目录
    • cd ..：打开上级目录

  • ls：查看当前文件夹下的所有文件夹

  • pwd：查看当前工作目录9的完整路径

  • df：查看当前磁盘使用情况

    • -h：使用K、M、G为单位显示
    • -a：显示所有文件系统的磁盘使用情况
    • -k：以k字节为单位显示
    • -T： 显示文件系统类型（T必须大写）
    • 无论在哪个文件下输入 df -h等命令，都会显示同样的信息，因为该命令显示的是当前磁盘的使用情况

  • du：查看当前目录空间使用情况

    • -h： 使用K、M、G为单位显示
    • -k：以KB(1024bytes)为单位输出
    • -m ：以MB为单位输出
    • -s：仅显示总计，只列出最后加总的值
    • 可以组合使用，比如打开usr文件夹，键入du -hs 或者du -sh 可以查看以GB为单位显示的总计
